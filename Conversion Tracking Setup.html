Implementation Example

Here's an example of how we implement conversion tracking code on your website:

<!-- Google Ads Conversion Tracking --> <script> gtag('event', 'conversion', { 'send_to': 'AW-XXXXXXXXX/XXXXXXXXXXXXXXX', 'value': 1.0, 'currency': 'USD', 'transaction_id': '' }); </script> <!-- Enhanced E-commerce Tracking --> <script> gtag('event', 'purchase', { 'transaction_id': '12345', 'value': 25.42, 'currency': 'USD', 'items': [{ 'item_id': 'SKU123', 'item_name': 'Product Name', 'category': 'Category', 'quantity': 1, 'price': 25.42 }] }); </script>
